## Overview
End-to-end data analytics project to analyze customer churn patterns using Python.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
IBM Telco Customer Churn Dataset

## Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Univariate & Bivariate Analysis
- Logistic Regression
- Random Forest Classifier

## Key Insights
- Month-to-month contracts have higher churn
- Customers with higher monthly charges are more likely to churn
- Long-term contracts reduce churn significantly

## Conclusion
This project helps businesses understand churn drivers and improve customer retention.
